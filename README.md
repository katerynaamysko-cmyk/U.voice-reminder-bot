# Telegram-бот "Нагадувач"

## 1. Створення бота

1. У Telegram напишіть **@BotFather**.
2. Команда `/newbot`, задайте ім'я та юзернейм.
3. Скопіюйте виданий **токен** — він знадобиться нижче.

## 2. Додавання бота в канал

- Якщо це **канал**: додайте бота як адміністратора (потрібні права
  "Публікація повідомлень").
- Якщо це **група/чат**: просто додайте бота як учасника.

## 3. Встановлення

```bash
pip install python-telegram-bot==21.6 --break-system-packages
```

## 4. Запуск

```bash
export BOT_TOKEN="ваш_токен_від_BotFather"
python bot.py
```

Бот працюватиме, поки запущений цей процес. Для постійної роботи 24/7
його треба розмістити на сервері (VPS, наприклад: DigitalOcean, Hetzner,
Amazon Lightsail) або хмарному сервісі (Railway, Render, Fly.io — мають
безкоштовні тарифи для невеликих ботів).

## 5. Як користуватись

1. У каналі/чаті напишіть команду, наприклад:
   `/remind Подайте заявку на конкурс до п'ятниці!`
2. Бот опублікує повідомлення з кнопками "Через 1 день / Через 3 дні /
   Через тиждень".
3. Користувач тисне потрібну кнопку.
4. **Якщо це перший раз** — бот попросить написати йому `/start` у
   приваті (обмеження Telegram: боти не можуть писати першими).
   Після цього кнопки працюють одразу.
5. У призначений час бот надішле нагадування в особисті повідомлення.

## Що можна доналаштувати

- **Варіанти нагадувань** — список `REMINDER_OPTIONS` у `bot.py`
  (зараз: 1 день / 3 дні / тиждень — можна додати свої).
- **Текст повідомлень** — легко змінити рядки з `reply_text`.
- **Нагадування за розкладом** (наприклад, "щопонеділка") — можна
  додати через `context.job_queue.run_daily(...)`.
- **Зберігання** — зараз SQLite-файл `reminders.db` поруч зі скриптом;
  нагадування переживають перезапуск бота.

## Обмеження, про які варто знати

- Бот **не може** писати першим користувачу, який жодного разу не
  писав боту в приват — це правило самого Telegram, обійти його не можна.
- Якщо користувач заблокує бота, нагадування просто не дійде (бот це
  коректно обробляє й не падає).
